YouTube Data API key (optional)

Without a key the extension still works in limited mode: duration totals for the videos visible in the playlist side panel, topic summaries, and the yt-dlp command generator. Adding a free API key enables the full playlist, likes, views and comment-sentiment features.

How to get a free API key
  1. Open Google Cloud Console.
  2. Create or pick a project.
  3. Open APIs & Services → Library and enable YouTube Data API v3.
  4. Open APIs & Services → Credentials, click Create credentials → API key.
  5. Paste it above and click Save.

Your key is stored only in this browser (chrome.storage.local) and is sent only to googleapis.com when you run an analysis. It never leaves your machine for any other destination.
